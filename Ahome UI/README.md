# A•HOME Riviera Maya

A modern, responsive website for A•HOME Riviera Maya - a real estate company specializing in properties in the Riviera Maya region of Mexico.

## Features

- **Modern Design**: Clean, professional layout with smooth animations
- **Responsive**: Fully responsive design that works on all devices
- **Smooth Scrolling**: Smooth navigation between sections
- **Interactive Elements**: Hover effects and animations throughout
- **Contact Form**: Functional contact form (ready for backend integration)
- **Property Showcase**: Beautiful property cards with images and details

## Structure

- `index.html` - Main HTML file with all sections
- `styles.css` - Complete styling with modern design
- `script.js` - JavaScript for interactivity and animations
- `README.md` - This file

## Sections

1. **Hero Section** - Eye-catching landing area with call-to-action buttons
2. **Features** - Key selling points of the service
3. **Properties** - Showcase of featured properties
4. **About** - Company information and statistics
5. **Services** - Services offered
6. **Contact** - Contact form and information
7. **Footer** - Additional links and information

## Customization

### Colors
The color scheme can be customized in `styles.css` by modifying the CSS variables:
- `--primary-color`: Main brand color
- `--secondary-color`: Accent color
- `--accent-color`: Secondary accent
- `--text-dark`: Dark text color
- `--text-light`: Light text color

### Content
All content can be easily modified in `index.html`. Simply update the text, images, and property information as needed.

### Images
The website uses Unsplash images. Replace the image URLs in the HTML with your own property images.
